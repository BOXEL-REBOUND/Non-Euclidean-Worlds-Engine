# NonEuclidean
A NonEuclidean rendering engine for Windows, written in C++ OpenGL.

## Source Code Dependencies
Add glew-2.1.0 to the main directory

## Controls
* **Mouse** - Look around
* **AWSD** - Movement
* **1 - 7** - Switch between different demo rooms
* **Alt + Enter** - Toggle Fullscreen
* **Esc** - Exit demo
